# Gestió de Material – Ins Camí de Mar

Sistema senzill per gestionar el préstec de materials dins d’un centre educatiu.  
Permet afegir, llistar, prestar i retornar materials.


## Estructura del projecte
Gestio-de-Material-Ins-Cami-de-Mar/ 
├── src/ → Codi font Java 
  │ └── com/institut/inventari/ 
  │ ├── Categoria.java 
  │ ├── Material.java 
  │ ├── Prestec.java 
  │ ├── InventariRepository.java 
  │ └── InventariService.java 
├── doc/ → Javadoc generat 
├── DocumentacioUsuari.pdf → Manual d’usuari 
├── DocumentacioTecnica.pdf → Manual tècnic 
└── README.md → Aquest fitxer


## Execució

1. Obre el projecte amb Eclipse o un IDE Java.
2. Executa el mètode `main()` dins de la classe `InventariService`.
3. Es mostraran per consola les operacions d’afegit, préstec i retorn de material.

## Documentació

- **Documentació tècnica** (`DocumentacioTecnica.pdf`):  
  Descripció de les classes, mètodes i arquitectura del sistema.

- **Documentació d’usuari** (`DocumentacioUsuari.pdf`):

- **Javadoc** (`doc/`):  
  Generat automàticament a partir dels comentaris del codi font.
